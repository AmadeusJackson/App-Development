# Extension Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create these icons using:

1. **Online Tools**:
   - [Favicon Generator](https://www.favicon-generator.org/)
   - [Icon Generator](https://icon.kitchen/)
   - [RealFaviconGenerator](https://realfavicongenerator.net/)

2. **Image Editors**:
   - Photoshop, GIMP, Figma, or any image editor
   - Create a square image (e.g., 512x512) and resize to the required sizes

3. **Quick Solution**:
   - Use any image (e.g., a lock icon, blocked YouTube logo)
   - Resize to the three required sizes
   - Save as PNG files with the exact names above

## Temporary Placeholder

Until you add proper icons, you can use any simple PNG images. The extension will work without icons, but Chrome will show a default placeholder icon.
